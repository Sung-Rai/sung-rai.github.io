Runelite Plugins

  • Thrall Highlighter - An official Runelite Plugin to hide and outline thralls

I kept finding myself fiddling with the entity hider plugin configuration when I wanted to toggle the visibilty of other players. I also saw a handful of people asking about how to hide and outline thralls, with the roundabout method of configuring two different plugins. So instead of waiting for someone else to make it, I made it.

I had a dream about this game when I was younger, and I really wanted to make it myself in the future. Mods

  • Balatro Four-Color Deck - When you need to differentiate the four suits.

The assets used in this mod was taken from the nexusmod user FabianPEKS upload of a Four-color deck. However, to install this you would need to mess with the game files directly without being able to switch back to the normal deck skin. So I made it work with Steammodded so I could enable/disable it easy through the config menu.
